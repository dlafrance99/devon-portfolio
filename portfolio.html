Portfolio

Full Stack apps

Save-the-Dish

Save the dish is a restaurant and menu review website. This app uses Node.js, Javascript, ejs, Passport, jQuery, mySQL, sequelize, express.js, and Heroku deployment.

Bob's Burger Bonanza

Bob's Burger Bonanza allows the user to add burgers to the menu, eat the burger, then eat again. This app uses MySQL, Node, Express, Handlebars, and ORM.

Front End Apps

Bearfoot-Voyager

Bearfoot-Voyager allows users to find information about a searched destination. This app uses front end design, APIs, AJAX calls, Skeleton framework, and input validation.

Train Scheduler

Train Scheduler allows users to check train schedules, as well as add a train of their own. This app uses firebase, Moment.js, and front end design.

Back End Apps

liri node

Liri allows users to search an artist, movie or song and it will return information about that subject. Liri uses Node, Spotify-api, axios, OMDB api, Bands in Town api, and Moment js.

Bamazon

Bamazon works as a mock storefront, allowing users to "buy items" from an inventory. This app uses Node, inquirer, and Mysql.
